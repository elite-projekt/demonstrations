## Phishing Demo

This component is a flask web API. It hosts fake phishing sites which the user encounters on their journey through ecs.

### Setup

#### Under Windows
```bash
cd 'Phishing Demo'
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
cd app\

python -m app

```

#### Under Linux
```bash
cd 'Phishing Demo'
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cd app/

python -m app

```
