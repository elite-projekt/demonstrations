2FA Einführung

Wir werden in diesem Beispiel die App Google Authenticator nutzen.

Der Google Authenticator ist sowohl für Android als auch iOS erhältlich.

Es gibt noch weitere kostenfreie 2FA Apps. Wenn Sie eine andere App nutzen möchten, informieren Sie sich vorher über die Sicherheit der App. Eine gute Grundvorraussetzung ist, wenn die App eine Open-Source-Lizenz besitzt. Weitere 2FA Apps finden Sie, indem Sie im Play Store oder im Apple App Store nach Authentifizierung suchen.

  1. Öffnen Sie Ihre Google Authenticator App auf Ihrem Smartphone.
  2. Beim ersten Starten sehen Sie einen Willkommensbildschirm. Klicken Sie auf Jetzt Starten.
  3. Klicken Sie nun auf QR-Code scannen.
  4. Die App fragt nach dem Zugriff auf Ihre Kamera. Bestätigen Sie diese Erlaubnis. Nun öffnet die App Ihre Kamera und Sie können den QR-Code auf der nächsten Seite scannen.
  5. Scannen Sie jetzt den angezeigten QR-Code auf dem Monitor.
  6. Es wird ein neues Konto angelegt und Sie sehen eine 6-stellige Zahl - der Sicherheitstoken Jeder Token ist 60 Sekunden lang gültig.
  7. Geben Sie den Code in das Eingabefeld ein und bestätigen Sie die Eingabe.
